API使用方法

请使用以下端点发送请求:

代理端点: /api/proxy

请求示例


fetch('/api/proxy', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: '写一篇关于人工智能的文章',
    user_id: '用户ID(可选)'
  })
})
.then(response => response.json())
.then(data => console.log(data));

响应格式


{
  "content": "AI生成的回复内容",
  "status": "success"
}

如果发生错误，将返回以下格式:


{
  "error": "错误描述",
  "status": "error"
}
